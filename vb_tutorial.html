A PDF Visual Basic Tutorial

A written VB.NET tutorial during my time at Crosskeys College, Newport Gwent (2020).

This 17-page Visual Basic Tutorial dives head first into the wonders of creating software applications using VB.NET, with straight-forward comemnts for anyone to pick up on quickly. It first gives an overview into the basics (if-statements, variables, data types, etc.), then we check out some of the tools avaliable for creating awesome and useful apps, and even learning the code to make them work.
